# 🎹 数字音乐作曲工作坊 Pro

一个功能强大的网页音乐创作应用，支持钢琴卷帘编辑、和弦自动伴奏、鼓点生成器、多轨道混音、音频效果器、MIDI键盘输入、作品录制和社区分享。

## ✨ 新功能特性

### 🎼 和弦自动伴奏
- **和弦库**: 支持13种常见和弦（C, Cm, D, Dm, Em, F, Fm, G, Gm, Am, Bdim等）
- **和弦行进**: 可视化和弦序列编辑，点击即可试听
- **随机生成**: 一键生成流行和弦进行（如C-Am-F-G, I-V-vi-IV等）
- **自动播放**: 播放时自动跟随和弦伴奏

### 🥁 鼓点生成器
- **4轨鼓机**: 底鼓、军鼓、踩镲、通鼓
- **5种预设**: 摇滚、流行、放克、爵士、电子
- **随机生成**: 一键生成随机鼓点
- **16步进音序器**: 可视化鼓点编辑

### 🎛️ 多轨道混音系统
- **3个独立通道**: 主旋律、和弦、鼓组
- **音量控制**: 每个通道独立音量滑块（0-100%）
- **静音/独奏**: 支持单个通道静音和独奏
- **实时生效**: 播放时可实时调整混音

### ✨ 音频效果器
- **混响 (Reverb)**:
  - 湿信号/干信号比例可调
  - 衰减时间可调（0.1-5秒）
  - 基于卷积的混响算法
  
- **延迟 (Delay)**:
  - 湿信号/干信号比例可调
  - 延迟时间可调（100-1000ms）
  - 反馈强度可调

### 👥 社区作品分享
- **作品发布**: 将作品发布到社区
- **作品浏览**: 浏览所有社区作品
- **排序功能**: 按热门/最新排序
- **作品改编 (Remix)**: 加载社区作品并进行改编创作
- **作品信息**: 显示作者、描述、播放数、点赞数

### 🎵 核心功能
- **钢琴卷帘编辑器**: 9个八度（C0-B8），108个琴键
- **音符编辑**: 拖拽移动、调整时长、Delete删除
- **Web Audio合成器**: 内置正弦波振荡器，带ADSR包络
- **MIDI键盘支持**: 自动检测外接MIDI设备，实时演奏
- **录音功能**: 将作品导出为WAV音频文件
- **拍号/调号**: 支持4/4, 3/4, 6/8拍号和多个调号
- **作品管理**: 保存/加载完整作品数据

## 🚀 快速开始

```bash
npm start
```

### 访问应用
在浏览器中打开: `http://localhost:3000`

## 📖 使用说明

### 和弦伴奏使用
1. 从下拉菜单选择和弦
2. 点击「添加」将和弦添加到序列
3. 点击序列中的和弦可试听
4. 点击「🎲 自动生成」随机生成流行和弦进行
5. 点击×删除不需要的和弦

### 鼓点编辑使用
1. 在4条鼓轨上点击步进格激活/取消鼓点
2. 点击「应用预设」选择不同风格的鼓点模板
3. 点击「🎲 随机」生成随机鼓点
4. 点击「清空」清除所有鼓点

### 混音台使用
1. 拖动滑块调整各通道音量
2. 点击「M」按钮静音单个通道
3. 点击「S」按钮独奏单个通道
4. 播放过程中可实时调整

### 效果器使用
1. 勾选「启用」开启对应效果器
2. 拖动「湿信号」滑块调整效果强度
3. 调整其他参数（混响衰减、延迟时间等）
4. 效果在播放时实时生效

### 社区功能使用
1. 点击「🌐 发布」将作品发布到社区
2. 点击「👥 社区」浏览所有作品
3. 点击「📂 加载」加载社区作品
4. 点击「🎨 改编」加载作品并进行改编创作

### 创建音符
1. 在钢琴卷帘网格上点击任意位置创建音符
2. 每个音符创建时会自动播放预览

### 编辑音符
- **移动音符**: 直接拖拽音符到目标位置
- **调整时长**: 拖拽音符右侧的白色调整手柄
- **删除音符**: 选中音符后按 Delete 键

### 播放控制
1. 点击 **▶ 播放** 开始播放
2. 点击 **⏸ 暂停** 暂停播放
3. 点击 **⏹ 停止** 停止并回到开头

### 录音
1. 点击 **⏺ 录音** 开始录音
2. 录音完成后自动下载WAV文件

### 保存/加载作品
1. 点击 **💾 保存** 输入作品名称保存
2. 点击 **📂 加载** 选择已保存的作品

### 使用MIDI键盘
1. 连接MIDI键盘到电脑
2. 应用会自动检测并显示"已连接"
3. 直接在MIDI键盘上演奏，音符会自动播放

## 🏗️ 项目结构

```
├── index.html              # 主页面
├── style.css               # 样式文件
├── app.js                  # 前端应用逻辑
├── server.js               # 后端服务器
├── package.json            # 项目配置
├── works/                  # 私有作品存储目录（自动创建）
└── community/              # 社区作品存储目录（自动创建）
```

## 🔧 API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/works | 获取私有作品列表 |
| POST | /api/works | 保存私有作品 |
| DELETE | /api/works/:name | 删除私有作品 |
| POST | /api/works/publish | 发布作品到社区 |
| GET | /api/works/community | 获取社区作品列表 |
| POST | /api/works/:name/like | 点赞作品 |
| POST | /api/works/:name/remix | 改编作品 |

## 🎯 系统要求

- **浏览器**: Chrome, Firefox, Safari, Edge (支持Web Audio API)
- **Node.js**: v14 或更高版本
- **MIDI支持**: 需要支持Web MIDI API的浏览器（Chrome最佳）

## 📝 注意事项

1. 首次使用录音功能需要授权麦克风访问权限
2. MIDI键盘需要浏览器支持Web MIDI API（Chrome最佳）
3. 作品默认保存在服务器的 `works/` 目录
4. 社区作品保存在 `community/` 目录
5. 如果服务器不可用，作品会自动保存到浏览器本地存储
